# EDR (Коллекция Реальности)

Минималистичный веб-интерфейс для представления музыкального кейса **EDR** (2025–2026).  
Создан только на HTML + CSS. Подходит для демонстрации процесса генерации треков через AI (Suno AI) и постобработки.

---

## 🎯 Цель
- Визуально соответствовать обложке: чёрно-белая архитектурная эстетика, рваный шрифт, геометрия.
- Обеспечить доступность: контраст ≥ 4.5:1, семантическая разметка, поддержка скринридеров.
- Сохранить UI-дружелюбность: читаемость, адаптивность, отсутствие мерцания/перегрузки.

---

## 🛠 Технологии
- **HTML5** — семантическая структура (`<section>`, `<ol>`, `aria-label`)
- **CSS3** — переменные, `backdrop-filter`, `clamp()` (опционально), адаптивные медиа-запросы
- **Шрифт**: Inter (system fallback — Segoe UI, sans-serif)
- **Без JS**, без внешних зависимостей

## 🖼 Особенности дизайна
| Элемент           | Реализация |
|-------------------|------------|
| Заголовок `EDR`   | Жирный uppercase + `text-shadow` + overlay-текстура (радиальные градиенты) — имитирует «рваную краску» |
| Фон              | Глубокий `#121212`, без градиентов — как на обложке |
| Кард             | `rgba(42, 42, 42, 0.9)` + `backdrop-filter: blur(10px)` — глубина без перегрузки |
| Плеер            | Чёткие границы, тень, полупрозрачный неоновый акцент (серо-голубой) — выделен, но не кричит |
| Кейс-блок        | Отступы, маркеры `—`, моноширинный `prompt-box` — как техническая документация |
