# DLTMZK — Release Card

Минималистичный веб-интерфейс для представления музыкальных релизов **СПЛЕТЕНИЯ** и **ТОЛЬКОБИТ** от артиста **DLTMZK** (2026).  
Создан на HTML + CSS + JS.

---

## 🎯 Цель
- Показать **только ключевую информацию** по релизам: обложка, название, артист, жанр, длительность, год.
- Сохранить визуальную связь с обложкой: рваный текст, морской пейзаж.
- Обеспечить доступность и адаптивность — для всех устройств и пользователей.
- Сохранить **минимализм** и **доступность**.

---

## ✨ Особенности
- **Hover-эффекты** для обложки и ключевых слов в метаданных — лёгкое всплытие и тонкая подсветка.
- **Градиент внизу карда** — создаёт ощущение глубины, как океанская впадина.
- **Индикатор активного плеера** — при фокусе (например, при навигации с клавиатуры) плеер подсвечивается неоновым акцентом.
- **Чистая семантика** — доступность, скринридеры, SEO.
- **Адаптивный дизайн** — корректное отображение на мобильных и десктопах.
- **Без лишних элементов** — только обложка, информация и плеер.
- **Текст песни** — скрыт по умолчанию, открывается по клику.
- **Кнопка "Скачать"** — с выбором формата (FLAC, MP3).
- **Разделитель между треками** — визуальное разделение.
- **Кнопки "Поделиться"** — копирование ссылки и отправка в ВКонтакте / Одноклассники / BlueSky.
- **Автоматическая остановка** одного трека при воспроизведении другого.
- **Метки треков** — указание на то, что трек создан с помощью нейросетей или без.
- **Сравнительная таблица стилей** — визуальное сравнение AI и Human-made треков.
- **Адаптивная таблица** — корректное отображение на мобильных устройствах.
- **Визуализация характеристик** — сравнение "атмосферности", "ритмичности" и др.
- **Система оценки треков** — возможность поставить оценку (звезды) каждому треку.

---

## 🛠 Технологии
- **HTML5** — семантическая разметка (`<article>`, `<audio>`, `<img>`, `<details>`, `<summary>`)
- **CSS3** — переменные, `backdrop-filter`, `:hover`, `::after`, `:focus-within`, адаптивные медиа-запросы
- **JavaScript** — копирование ссылки, генерация URL поделки, остановка других треков, система оценки
- **Шрифт**: Inter (system fallback — Segoe UI, sans-serif)

## 🖼 Особенности дизайна
| Элемент                   | Реализация |
|---------------------------|------------|
| Фон                       | Глубокий `#0a141d` + subtle radial gradients — имитация воды |
| Кард                      | `rgba(12,26,37,0.85)` + `backdrop-filter: blur(10px)` — как «остров в волне» |
| Заголовок `СПЛЕТЕНИЯ`     | `-webkit-text-stroke` + `text-shadow` — чёткий, с обводкой |
| Метаданные                | Однострочное отображение, `text-overflow: ellipsis`, подчёркивание при наведении |
| Плеер                     | Чёткие грани, тень, полупрозрачный фон — выделен, но не доминирует |
| Градиент внизу            | `linear-gradient` — усиливает ощущение глубины |
| Обложка                   | `transform: translateY(-4px)` при наведении — лёгкое всплытие |
| Индикатор активного плеера| `:focus-within` — подсветка рамки при фокусе (доступность) |
| Разделитель треков        | `.track-separator` — визуальный элемент между треками |
| Метки треков              | `.ai-badge`, `.human-badge` — различие AI/Human-made |
| Сравнительная таблица     | `.comparison-list` — адаптивная верстка, читаема на моб. |
| Визуализация характеристик| `.comparison-bars-section` — CSS-бары для наглядности |
| Система оценки            | `.rating-section` — звёзды, сохранение в `localStorage` |
