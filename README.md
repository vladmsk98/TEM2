# TEM2 — Release Card

Минималистичный веб-интерфейс для представления музыкального релиза **TEM2** (2026).  
Создан только на HTML + CSS.

---

## 🎯 Цель
- Показать **только ключевую информацию** по релизу: обложка, название, артист, жанр, длительность, год.
- Сохранить визуальную связь с обложкой: чёрно-белая волна, рваный текст, морская глубина.
- Обеспечить доступность и адаптивность — для всех устройств и пользователей.
- Добавить аккуратные, почти незаметные улучшения, не нарушая минимализм.

---

## ✨ Особенности
- **Два трека** на одной странице — структурированы с помощью визуального **разделителя**.
- **Hover-эффекты** для обложки и ключевых слов в метаданных — лёгкое всплытие и тонкая подсветка.
- **Градиент внизу карда** — создаёт ощущение глубины, как океанская впадина.
- **Индикатор активного плеера** — при фокусе (например, при навигации с клавиатуры) плеер подсвечивается неоновым акцентом.
- **Чистая семантика** — доступность, скринридеры, SEO.
- **Адаптивный дизайн** — корректное отображение на мобильных и десктопах.

---

## 🛠 Технологии
- **HTML5** — семантическая разметка (`<article>`, `<audio>`, `<img>`)
- **CSS3** — переменные, `backdrop-filter`, `:hover`, `::after`, `:focus-within`, адаптивные медиа-запросы
- **Шрифт**: Inter (system fallback — Segoe UI, sans-serif)
- **Без JS**, без внешних зависимостей

## 🖼 Особенности дизайна
| Элемент                   | Реализация |
|---------------------------|------------|
| Фон                       | Глубокий `#0a141d` + subtle radial gradients — имитация воды |
| Кард                      | `rgba(12,26,37,0.85)` + `backdrop-filter: blur(10px)` — как «остров в волне» |
| Заголовок `TEM2`          | `-webkit-text-stroke` + `text-shadow` — точная имитация обводки на обложке |
| Метаданные                | Однострочное отображение, `text-overflow: ellipsis`, подчёркивание при наведении |
| Плеер                     | Чёткие грани, тень, полупрозрачный фон — выделен, но не доминирует |
| Градиент внизу            | `linear-gradient` — усиливает ощущение глубины |
| Обложка                   | `transform: translateY(-4px)` при наведении — лёгкое всплытие |
| Разделитель               | Центральная точка `• • •` — визуально разделяет треки |
| Индикатор активного плеера| `:focus-within` — подсветка рамки при фокусе (доступность) |
